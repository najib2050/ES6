WHAY USE PROMISE IN JS

-promise can mitigate the CALLBACK HELL; the situation where code marches to the right faster than it marches forword.

-it allows you to associate handlers with an asynchronouse action's eventual success value or failure reasons.

promises make code easier to read and understand, especially when dealing with multiple asynchronouse tasks.

provide a centralized handler for error management simplifying error handling.
