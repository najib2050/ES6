PROMISE METHODS

Various methods available to the promise object are as follows:

1: all(iterable) => waits for all promises to be resolved or any one to be rejected. 2: finally() =>appends a handler to the promise. 3: any(iterable) =>return the promise value when any one of the promise is fulfilled. 4: rece(iterable) =>waits until any of the promises is resolved or rejected. 5: reject (reason) =>return anew promises object that is rejected for the given reason. 6: resolve(value) =>returns a new promise object that resove with the given value.
